# Introduction
1. Developed by Sun Microsystem in 1991,later acquired by Oracle Corporation.
2. Java was developed by **James Gosling** and his team at **Sun Microsystems** in **1995**.
3.  It was initially created as part of a project called **"Green Project"** and was designed for interactive television.
4.  The language was originally named **Oak** (after an oak tree outside Gosling's office), but the name was later changed to **Java**,
   inspired by Java coffee, as a nod to its simplicity and energy.
5. Sun Microsystems was later acquired by **Oracle Corporation** in 2010, which now owns and maintains Java.
6. Java is a popular, platform-independent programming language designed for flexibility, reliability, and simplicity.
7. It is widely used for web, desktop, mobile applications, and enterprise software.
8. Key Features
- Object-Oriented: Everything is based on objects and classes.
- Platform-Independent: Write once, run anywhere (via JVM).
- Simple & Secure: Easy-to-learn syntax with robust security.
- Rich Libraries: Extensive APIs for various functionalities.
